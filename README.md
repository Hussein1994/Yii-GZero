Yii Ground-Zero App

Yii Grand-Zero Ready-to-code application template

- URL manager.
- SQL database.
- Local config file.
- rbac system.
- multi-language.

- site settings, admins, Google analytics.
- Text helper.
- mobile detector.
- CSS/JS Compressor (Docs: http://www.yiiframework.com/extension/eclientscript/).
- Yii debug toolbar.
- Google analytics.
